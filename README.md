# educalo

## What is it ?

*educalo* is a custom software to control the *educalo* hardware, a low-cost calorimeter dedicated to education and experimental investigation of cementitious materials

It is developed in the Center of Excellence for Low-Carbon Concrete, Université de Sherbrooke, Qc, Canada by [Fabien Georget](https://orcid.org/0000-0002-8212-9152).

## How to use it ?

### Hardware 

The hardware is based on the [RaspberryPi pico 2](https://www.raspberrypi.com/documentation/microcontrollers/pico-series.html#pico2) and can be replicated for less than 100 canadian dollars. The schematic will be made available soon.

### Software

*educalo* is developed using the [Zephyr RTOS](https://zephyrproject.org/). The code is currently organized as a [T1 topology](https://docs.zephyrproject.org/latest/develop/west/workspaces.html#t1-star-topology-zephyr-is-the-manifest-repository), and require a valid zephyr installation. It is tested with the zephyr 4.3+ version.
